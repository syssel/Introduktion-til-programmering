  • En .py-fil kan i python blive brugt på to måder:
    • Hovedprogram ("main"-program)

    • Værdien der er assignet er forskellig alt efter om py-filen er importeret (et modul) eller kører som hovedprogram (når i klikker 'run' i Idle)

				Returns a bibliography entry from the bibliography 
				data given by the parameters of the function.
				
				The returned will have the following format:
				LastName, FirstName. Title. Proceedings. StartP-EndP. (Year)
				"""

				# Format bib entry
				Name_format = LastName + ", " + FirstName
				Title_format = '"' + Title + '"'
				Proceedings_format = Proceedings
				Pages_format = str(StartP) + "-" + str(EndP)
				Year_format = "("+str(Year)+")"

				# Delimiter
				delimiter = ". "

				return(Name_format + delimiter + Title_format + delimiter +
						Proceedings_format + delimiter + Pages_format +
						delimiter + Year_format)

		def splitlist(L):
			""" list -> list(list, list)
			Splits the input list, L, into two sublists of (if possible) equal size.
			he function returns one list with the sublists nested.
			If the length of L is uneven, the first sublist should be shorther than the second
			"""

			sublists = [[], []]
			split_at = int(len(L)/2)
			sublists[0] = L[0:split_at]
			sublists[1] = L[split_at:]

			return sublists

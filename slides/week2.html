    • tager 2 argumenter - højde og vægt
    • returnerer et BMI-tal
  3. Giv funktionen en doc-string
  4. Brug funktionen til at beregne BMI-tallet for en person som vejer 75 kilo og er 170 cm høj

1.1 Funktion til at beregne BMI


			def BMICalculator(height, weight):
				""" (float, float ) -> float
			            Returns the BMI calculated from height and weight
			            parameters using the formula weight/height**2.
				"""

				return weight/height**2

			print(BMICalculator(1.73, 75))

2.2 Automatisk formattering af litteraturliste


			def BibliographyGenerator(FirstName, LastName, Year, Title, 
										Proceedings, StartP, EndP):
				""" (str, str, str, str, str, int, int) -> str
				Returns a bibliography entry from the bibliography 
				data given by the parameters of the function.
				"""

				# Format bib entry
				Name_format = LastName + ", " + FirstName
				Title_format = '"' + Title + '"'
				Proceedings_format = Proceedings
				Pages_format = str(StartP) + "-" + str(EndP)
				Year_format = "("+str(Year)+")"

				# Delimiter
				delimiter = ". "

				return(Name_format + delimiter + Title_format + delimiter +
						Proceedings_format + delimiter + Pages_format +
						delimiter + Year_format)
